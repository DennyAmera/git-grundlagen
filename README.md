# git-grundlagen
# Aufgabe 8.
Ein lokales Repository ist ein Projekt bzw eine kopie eines Projektes,
auf dem lokalen Computer. Ein remote Repository ist eine version des Projekts, die auf
einem externen Server (z.B. GitHub) gehostet wird. Mit dem lokalen Repository kann
unabhaengig gearbeitet und getestet werden, bevor die Dateien auf das remote Repository
geladen werden, wo das Entwickler-Team gemainsam arbeitet.
